# fzf-kill
An easy to hack script to kill stuff. 

## How to install

    yay -S fzf-kill

Or manually put the file into ~/.local/bin/fzf-kill

## How to use

run it like

    fzf-kill
or

    # More cluttered, but also faster.
    fzf-kill --detailed
